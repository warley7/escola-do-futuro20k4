Trabalhando com listas

Listas ordenada

Listas ordenadas são lista que servem para organizar iténs que possue, sequencial lógico. A lista possue a TAG OL e os iténs tem a RAG LI

  1. primeiro itém
  2. Segundo itém
  3. Terceiro itém
  4. quarto itém

Lista não ordenada

Lista não ordenadas são listas que não possuem uma sequência lógica, geralmente utiliza marcadores e usa-se a TAG UL para lista e LI para o itém

  • primeiro itém
  • Segundo itém
  • Terceiro itém
  • quarto itém
